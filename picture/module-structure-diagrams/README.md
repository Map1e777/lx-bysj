# 模块结构图 Mermaid 源码

本目录保存各功能模块的 Mermaid 模块结构图源码，便于论文插图和统一渲染。

文件列表：

- `01-用户认证模块结构图.mmd`
- `02-文档管理模块结构图.mmd`
- `03-版本管理模块结构图.mmd`
- `04-协作管理模块结构图.mmd`
- `05-评论交流模块结构图.mmd`
- `06-消息通知模块结构图.mmd`
- `07-文档分享模块结构图.mmd`
- `08-组织管理模块结构图.mmd`
- `09-平台管理模块结构图.mmd`

所有图均采用自顶向下结构，并配置了 Mermaid 间距参数：

- `nodeSpacing: 60`
- `rankSpacing: 80`
- `curve: linear`
