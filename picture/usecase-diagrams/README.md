# 用例图源码

目录中的文件为论文/文档可直接使用的 PlantUML 用例图源码。

文件说明：

- `01-user-auth.puml`：用户认证功能用例图
- `02-document-management.puml`：文档管理功能用例图
- `03-version-management.puml`：版本管理功能用例图
- `04-collaboration-management.puml`：协作管理功能用例图
- `05-comment-communication.puml`：评论交流功能用例图
- `06-notification-management.puml`：消息通知功能用例图
- `07-document-sharing.puml`：文档分享功能用例图
- `08-organization-management.puml`：组织管理功能用例图
- `09-platform-management.puml`：平台管理功能用例图

所有文件均已包含：

- `skinparam linetype ortho`

用于让连线尽量保持横平竖直。
